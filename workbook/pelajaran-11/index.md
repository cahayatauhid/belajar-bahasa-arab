# Workbook Pelajaran 11 --- Isim Kana dan Khabar Inna

Workbook ini bukan PR yang harus dikejar selesai, melainkan **panduan untuk lebih sadar saat melakukan ibadah yang sudah rutin**. Yang berubah minggu ini: setiap kali Anda dengar atau baca \ar{إِنَّ} atau \ar{كَانَ} di Quran/dzikir, mata Anda mulai mengidentifikasi **isim** (yang berperan seperti subjek) dan **khabar** (yang berperan menjelaskannya), serta keadaan i'rabnya yang **berbeda dari mubtada-khabar biasa**.

> **Adab pengamatan**: pengamatan dilakukan di luar shalat. Saat shalat, fokus utama tetap khusyu' kepada Allah.

## Tantangan Vocabulary Mingguan

Target minggu ini: **kumpulkan 5 ayat** yang memuat \ar{إِنَّ} atau \ar{كَانَ}, dengan identifikasi isim dan khabar mereka.

\ar{إِنَّ} adalah salah satu kata paling tersebar di Quran --- hampir setiap halaman punya beberapa contoh. Strategi:

**Strategi 1: Awal Surat / Awal Ayat.**

Banyak surat dimulai dengan \ar{إِنَّ}:

- Al-Inshirah (94:5): \ar{فَإِنَّ مَعَ الْعُسْرِ يُسْرًا}
- Al-Qadr (97:1): \ar{إِنَّآ أَنزَلْنَٰهُ فِى لَيْلَةِ ٱلْقَدْرِ}
- Al-Kawthar (108:1): \ar{إِنَّآ أَعْطَيْنَٰكَ ٱلْكَوْثَرَ}
- Al-'Ashr (103:2): \ar{إِنَّ الْإِنْسَانَ لَفِى خُسْرٍ}

**Strategi 2: Akhir Ayat.**

Banyak ayat tertutup dengan \ar{إِنَّ اللَّهَ + sifat Allah}, mis. \ar{إِنَّ اللَّهَ غَفُورٌ رَحِيمٌ}.

### Catatan Vocabulary Anda

| Sumber Ayat | Nawasikh (\ar{إِنَّ} / \ar{كَانَ}) | Isim (apa) | Khabar (apa) |
|-------------|-------------------------------------|-------------|---------------|
|             |                                     |             |               |
|             |                                     |             |               |
|             |                                     |             |               |
|             |                                     |             |               |
|             |                                     |             |               |

Tip: **isim inna manshub** (fathah/fathatain), **khabar inna marfu'** (dhammah/dhammatain). Untuk kana sebaliknya: **isim kana marfu'**, **khabar kana manshub**.

## Lab Shalat: Memetakan \ar{إِنَّ} di Tasyahud dan Surat Pendek

### Misi 1 --- Tasyahud (At-Tahiyyat)

\ayat{أَشْهَدُ أَنْ لَا إِلَٰهَ إِلَّا اللَّهُ وَأَشْهَدُ أَنَّ مُحَمَّدًا رَسُولُ اللَّهِ}

Konsentrasi pada bagian kedua: \ar{أَنَّ مُحَمَّدًا رَسُولُ اللَّهِ}. \ar{أَنَّ} adalah saudara \ar{إِنَّ} dengan fungsi sama (me-nashb isim, me-rafa' khabar).

| Kata           | Jabatan        | I'rab |
|----------------|----------------|-------|
| \ar{أَنَّ}      | harf taukid    | ---   |
| \ar{مُحَمَّدًا} | isim anna      | nashb (fathatain) |
| \ar{رَسُولُ}    | khabar anna    | rafa' (dhammah)   |
| \ar{اللَّهِ}    | mudhaf ilaih untuk \ar{رَسُولُ} | jarr (kasrah) |

Refleksi: tasyahud berisi **dua kalimat syahadat** dengan struktur grammar yang berbeda:

- Syahadat pertama: \ar{لَا إِلَٰهَ إِلَّا اللَّهُ} --- pakai \ar{لَا} **nafi al-jins** (akan didetailkan di Pelajaran 17), bukan \ar{إِنَّ}.
- Syahadat kedua: \ar{أَنَّ مُحَمَّدًا رَسُولُ اللَّهِ} --- pakai \ar{أَنَّ} (taukid).

Mengapa beda strukturnya? (Hint: pernyataan tauhid butuh **negasi mutlak** semua tuhan kecuali Allah; pernyataan kerasulan butuh **penegasan** akan kerasulan Muhammad. Pilihan grammar mengikuti makna.)

### Misi 2 --- Surat Al-Kawthar 108:3

\ayat{إِنَّ شَانِئَكَ هُوَ الْأَبْتَرُ}

Identifikasi:

| Kata             | Jabatan          | I'rab          |
|------------------|------------------|----------------|
| \ar{إِنَّ}        |                  | ---            |
| \ar{شَانِئَكَ}    |                  |                |
| \ar{هُوَ}        | (mubtada baru, fi mahalli rafa) | ---  |
| \ar{الْأَبْتَرُ}  | (khabar dari \ar{هُوَ})| rafa'         |

Catatan: \ar{شَانِئَكَ} adalah isim inna --- fathah pada \ar{شَانِئَ} + dhamir muttashil \ar{ـكَ}. Khabar inna di sini bukan kata sederhana, tetapi **jumlah ismiyyah baru** (\ar{هُوَ الْأَبْتَرُ}). Konsep "khabar berbentuk jumlah" akan didetailkan di Level 2.

### Misi 3 --- Doa Sayyidul Istighfar

\ar{اللَّهُمَّ أَنْتَ رَبِّي لَا إِلَٰهَ إِلَّا أَنْتَ خَلَقْتَنِي وَأَنَا عَبْدُكَ وَأَنَا عَلَى عَهْدِكَ وَوَعْدِكَ مَا اسْتَطَعْتُ}

Doa ini banyak kalimat. Cek: ada \ar{إِنَّ} atau \ar{كَانَ}? \_\_\_\_ (Jawaban: tidak --- doa ini pakai jumlah ismiyyah biasa dengan dhamir munfashil sebagai mubtada.)

Refleksi: tidak semua doa pakai nawasikh. Tapi pelajaran sekarang: **mengenali kapan ada nawasikh** dan **kapan tidak ada**.

## Lab Dzikir: \ar{كَانَ} dalam Sejarah Islam

Dzikir-dzikir tertentu memuat fi'il \ar{كَانَ}. Salah satu contoh: shalawat. Ada banyak versi shalawat; salah satu yang panjang:

\ar{اللَّهُمَّ صَلِّ عَلَى مُحَمَّدٍ وَعَلَى آلِ مُحَمَّدٍ كَمَا صَلَّيْتَ عَلَى إِبْرَاهِيمَ وَعَلَى آلِ إِبْرَاهِيمَ إِنَّكَ حَمِيدٌ مَجِيدٌ}

Cek bagian akhir: \ar{إِنَّكَ حَمِيدٌ مَجِيدٌ}.

| Kata           | Jabatan       | I'rab           |
|----------------|---------------|-----------------|
| \ar{إِنَّ}      |               | ---             |
| \ar{ـكَ}        | isim inna (dhamir muttashil) | mabni, fi mahalli **nashb** |
| \ar{حَمِيدٌ}    | khabar inna   |                 |
| \ar{مَجِيدٌ}    | (khabar inna kedua, atau na'at) |     |

Refleksi: kalimat penutup shalawat ini Anda baca **5+ kali sehari** di shalat. Apakah baru kali ini Anda **memetakan** isim inna dan khabar inna-nya?

## Eksplorasi Mendalam: \ar{إِنَّ الْإِنْسَانَ لَفِي خُسْرٍ}

\ayat{وَالْعَصْرِ ۝ إِنَّ الْإِنْسَانَ لَفِى خُسْرٍ ۝ إِلَّا الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ وَتَوَاصَوْا بِالْحَقِّ وَتَوَاصَوْا بِالصَّبْرِ}

> "Demi masa. Sesungguhnya manusia benar-benar dalam kerugian, kecuali orang-orang yang beriman dan beramal shalih..." (QS Al-'Ashr 103)

Ayat 2 punya struktur \ar{إِنَّ + isim + khabar}. Analisis lengkap:

| Kata             | Jabatan          | I'rab          |
|------------------|------------------|----------------|
| \ar{إِنَّ}        | harf taukid      | ---            |
| \ar{الْإِنْسَانَ}| isim inna        | nashb (fathah) |
| \ar{لَ}          | lam ibtida (lam taukid) | --- (la mahalla lahu) |
| \ar{فِى}         | harf jarr        | ---            |
| \ar{خُسْرٍ}       | majrur dengan harf jarr | jarr (kasratain) |

Khabar inna di sini = \ar{لَفِى خُسْرٍ} (syibhul jumlah --- jarr-majrur). Khabar bisa berbentuk syibhul jumlah, sebagaimana sudah disinggung di Pelajaran 8.

Refleksi:

- Mengapa Allah pakai \ar{إِنَّ + لَ}, dua penegasan sekaligus? (Hint: untuk **menegaskan secara mutlak** bahwa kerugian itu **kepastian**, bukan kemungkinan.)

## Lab Quran: \ar{كَانَ} di Ayat-Ayat Sirah

\ar{كَانَ} sangat sering muncul di ayat-ayat sirah/cerita untuk menyatakan **keadaan masa lampau**. Contoh paling tersebar:

\ayat{كَانَ النَّاسُ أُمَّةً وَاحِدَةً}

> "Dahulu manusia adalah umat yang satu" (QS Al-Baqarah 2:213)

Analisis:

| Kata           | Jabatan       | I'rab          |
|----------------|---------------|----------------|
| \ar{كَانَ}     | fi'il madhi naqish | ---       |
| \ar{النَّاسُ}  | isim kana     | rafa' (dhammah)|
| \ar{أُمَّةً}    | khabar kana   | nashb (fathatain) |
| \ar{وَاحِدَةً}  | na'at untuk \ar{أُمَّةً} | nashb (mengikuti) |
