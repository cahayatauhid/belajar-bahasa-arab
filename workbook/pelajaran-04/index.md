# Workbook Pelajaran 4 --- Tulisan Arab dan Konsep Kalam

Workbook ini menggabungkan **dua kategori praktik**: drill menulis huruf hijaiyah dengan tangan (sebelum masuk ke konsep grammar di Pelajaran 5), dan latihan membedakan **kalam** (kalimat sempurna) dari **kalimah** (kata) di teks-teks yang Anda kenal. Targetnya: minggu ini tangan Anda terbiasa menulis huruf Arab dengan benar, dan mata Anda terbiasa **memenggal** ayat menjadi unit-unit kata.

> **Adab pengamatan**: drill menulis dilakukan di luar shalat, di buku tulis bergaris atau buku catatan. Saat shalat, fokus utama tetap khusyu' kepada Allah.

## Lab Menulis: Drill Huruf Hijaiyah

Tujuan: tangan Anda terbiasa menulis huruf Arab dalam **empat bentuk** (berdiri sendiri, awal, tengah, akhir) di buku tulis bergaris.

Siapkan: buku tulis bergaris (yang punya tiga garis sejajar untuk huruf Latin --- yang dulu dipakai SD), pensil/pulpen.

### Drill 1 --- Empat Bentuk Setiap Huruf (di Buku Bergaris)

Pilih **lima huruf** dari hijaiyah. Untuk masing-masing, tulis di buku bergaris dalam empat bentuk:

| Huruf      | Berdiri Sendiri | Awal | Tengah | Akhir |
|------------|-----------------|------|--------|-------|
| \ar{ب}     | \ar{ب}          | \ar{بـ} | \ar{ـبـ} | \ar{ـب} |
| \ar{م}     |                 |      |        |       |
| \ar{ع}     |                 |      |        |       |
| \ar{ك}     |                 |      |        |       |
| \ar{ج}     |                 |      |        |       |

Saat menulis, perhatikan **zona** (atas/tengah/bawah):

- Huruf \ar{ا}, \ar{ل}, \ar{ك}, \ar{ط}, \ar{ظ} menjulang ke **zona atas**.
- Huruf \ar{ج}, \ar{ح}, \ar{خ}, \ar{ر}, \ar{ز} berekor turun ke **zona bawah** (selalu).
- Mayoritas huruf duduk di **zona tengah**.

Ulang setiap huruf 3 baris di buku bergaris. Tujuan: gerakan menjadi otomatis, bukan dipikirkan.

### Drill 2 --- Enam Huruf yang Tidak Bisa Disambung Setelahnya

Enam huruf ini punya aturan khusus: TIDAK bisa disambung dengan huruf setelahnya. Huruf setelahnya harus mulai sebagai bentuk awal.

\ar{ا}, \ar{د}, \ar{ذ}, \ar{ر}, \ar{ز}, \ar{و}

Tulis lima kata berikut, perhatikan **putus-sambung** huruf-hurufnya:

| Kata            | Cara baca | Putus di mana?           |
|-----------------|-----------|--------------------------|
| \ar{دَارٌ}      | dar       | setelah \ar{د} → \ar{ا} mulai bentuk awal |
| \ar{رَزَقَ}     | razaqa    | setelah \ar{ر} dan \ar{ز}                  |
| \ar{وَقَفَ}     | waqafa    | setelah \ar{و}                              |
| \ar{رَوَى}      | rawa      | setelah \ar{ر} dan \ar{و}                  |
| \ar{دُعَاءٌ}    | du'a      | setelah \ar{د}                              |

Tulis ulang dengan tangan di buku bergaris. Cek hasilnya: apakah memang ada "celah" di tempat yang dimaksud?

## Lab Quran: Memilah Al-Fatihah jadi Kalimah

Pelajaran 4 mengenalkan dua istilah dasar:

- **Kalam** = kalimat yang sempurna maknanya
- **Kalimah** = kata (unit ucapan bermakna tunggal)

Sekarang, ambil Al-Fatihah dan **pilah** ayat-ayatnya menjadi kalimah-kalimah.

### Latihan: Al-Fatihah Ayat 1--2

\ayat{بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ٱلْحَمْدُ لِلَّهِ رَبِّ ٱلْعَٰلَمِينَ}

Pisahkan jadi kalimah (kata-kata):

| No | Kalimah         | Apa Anda kenal artinya?  |
|----|-----------------|--------------------------|
| 1  | \ar{بِ}         | (dengan)                 |
| 2  | \ar{اسْمِ}      |                          |
| 3  | \ar{اللَّهِ}    |                          |
| 4  | \ar{الرَّحْمَٰنِ} |                          |
| 5  | \ar{الرَّحِيمِ}  |                          |
| 6  | \ar{ٱلْحَمْدُ}   |                          |
| 7  | \ar{لِ}         | (untuk/bagi)             |
| 8  | \ar{اللَّهِ}    |                          |
| 9  | \ar{رَبِّ}      |                          |
| 10 | \ar{ٱلْعَٰلَمِينَ} |                          |

Total: 10 kalimah dalam dua ayat.

Pertanyaan: 

- \ar{بِسْمِ اللَّهِ} adalah **bukan kalam** (kalimat sempurna), karena maknanya menggantung ("dengan nama Allah... lalu apa?"). Yang membuat \ar{بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ ٱلْحَمْدُ لِلَّهِ رَبِّ ٱلْعَٰلَمِينَ} sebagai kalam adalah **gabungan** dengan ayat 2 yang punya \ar{ٱلْحَمْدُ لِلَّهِ} sebagai jumlah ismiyyah lengkap. Apakah ini sesuai dengan pengertian kalam yang Anda pelajari di textbook?

### Latihan: Al-Fatihah Ayat 5

\ayat{إِيَّاكَ نَعْبُدُ وَإِيَّاكَ نَسْتَعِينُ}

Pisahkan jadi kalimah:

| No | Kalimah          | Bisa berdiri sendiri jadi kalam? |
|----|------------------|------------------------------------|
| 1  | \ar{إِيَّاكَ}     | tidak (kata ganti orang)         |
| 2  | \ar{نَعْبُدُ}     | ya (kalam: "kami menyembah")     |
| 3  | \ar{وَ}           | tidak (harf athaf)                |
| 4  | \ar{إِيَّاكَ}     |                                    |
| 5  | \ar{نَسْتَعِينُ}  |                                    |

\ar{نَعْبُدُ} sendiri adalah kalam karena fi'il punya dhamir mustatir ("kami") yang tersirat. Tapi keseluruhan \ar{إِيَّاكَ نَعْبُدُ} memberi makna lebih lengkap dengan eksklusivitas.

## Eksplorasi Mendalam: Definisi Klasik Kalam

\ar{اللَّفْظُ الْمُرَكَّبُ الْمُفِيدُ بِالْوَضْعِ}

> "Lafaz yang tersusun, yang memberi faedah, berdasarkan ketetapan bahasa."

Empat unsur dari definisi ini:

1. **Lafaz** --- ucapan, bukan tulisan/isyarat
2. **Tersusun** --- minimal dua kalimah
3. **Memberi faedah** --- maknanya sempurna
4. **Berdasarkan ketetapan bahasa** --- mengikuti kaidah, bukan ucapan acak

Untuk masing-masing ucapan berikut, tentukan **kategori mana yang tidak terpenuhi** sehingga dia bukan kalam:

| Ucapan                              | Bukan kalam karena... | Unsur yang tidak terpenuhi |
|-------------------------------------|-----------------------|-----------------------------|
| Menulis "بسم الله" tanpa membaca    |                       | (lafaz)                     |
| \ar{زَيْدٌ}                         |                       | (tersusun)                  |
| \ar{إِذَا قَامَ زَيْدٌ}             |                       | (memberi faedah)            |
| \ar{رَجُلٌ كِتَابٌ بَيْتٌ}         |                       | (ketetapan bahasa)          |

## Lab Tulis Tangan: Surat Al-Ikhlas

Sebagai penutup praktik menulis minggu ini, **salin** Surat Al-Ikhlas dengan tangan Anda di buku tulis bergaris (boleh dengan harakat, atau tanpa harakat untuk yang lebih advanced):

\ayat{قُلْ هُوَ اللَّهُ أَحَدٌ ۝ اللَّهُ الصَّمَدُ ۝ لَمْ يَلِدْ وَلَمْ يُولَدْ ۝ وَلَمْ يَكُنْ لَهُ كُفُوًا أَحَدٌ}

Tulis ulang **3 kali**. Setiap kali, perhatikan satu hal yang berbeda:

- **Kali ke-1**: fokus pada **zona** (atas/tengah/bawah). Apakah huruf \ar{ل} di \ar{اللَّهُ} sudah menjulang cukup tinggi? Apakah \ar{ج} atau \ar{ح} (kalau ada) berekor turun?
- **Kali ke-2**: fokus pada **sambung-putus**. Setelah huruf \ar{د} (mis. di \ar{أَحَدٌ}), apakah huruf berikutnya betul-betul mulai sebagai bentuk baru?
- **Kali ke-3**: tutup mushaf, tulis dari hafalan. Periksa kesalahan dengan membandingkan ke mushaf.
