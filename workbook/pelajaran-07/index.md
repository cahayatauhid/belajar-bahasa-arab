# Workbook Pelajaran 7 --- Empat Keadaan I'rab

Workbook ini bukan PR yang harus dikejar selesai, melainkan **panduan untuk lebih sadar saat melakukan ibadah yang sudah rutin**. Yang berubah minggu ini: setiap kata mu'rab yang Anda lafalkan, mata Anda mulai **mengkategorikan keadaan i'rabnya** ke salah satu dari empat: rafa', nashb, jarr, atau jazm. Inilah penutup Blok 1 (Pengantar dan Motivasi) sebelum Blok 2 (Marfu'at) dimulai.

> **Adab pengamatan**: saat shalat, fokus utama tetap khusyu' kepada Allah. Pengamatan keadaan i'rab dilakukan **setelah salam**, atau saat membaca ulang ayat di luar shalat untuk persiapan/refleksi.

3. Apa beda **mu'rab**, **mabni fi mahalli**, dan **la mahalla lahu**?

## Tantangan Vocabulary Mingguan

Target minggu ini: **kumpulkan 5 kata baru** dengan fokus tematik **isim yang muncul dalam berbagai keadaan i'rab di Quran**.

### Strategi: Cari Kata yang Sama dengan I'rab Berbeda

Sebagian kata sangat sering muncul di Quran dengan i'rab yang berubah-ubah. Ambil contoh paling tersebar:

- \ar{اللَّهُ} (rafa'), \ar{اللَّهَ} (nashb), \ar{اللَّهِ} (jarr) --- semua tiga muncul berkali-kali
- \ar{رَبٌّ} dengan variasi
- \ar{كِتَابٌ} dengan variasi (mufrad, mu'arraf, dst.)

Cari di tilawah Anda atau di surat hafalan, lalu catat:

| Kata          | Bentuk Rafa' | Bentuk Nashb | Bentuk Jarr | Sumber yang Anda Temui |
|---------------|--------------|--------------|-------------|------------------------|
|               |              |              |             |                        |
|               |              |              |             |                        |
|               |              |              |             |                        |
|               |              |              |             |                        |
|               |              |              |             |                        |

Kalau hanya ketemu satu bentuk, isi yang ada saja. Latihan ini melatih mata Anda mengenali **satu kata** dalam **berbagai harakat akhir**.

## Lab Shalat: Pemetaan I'rab di Tasyahud

Tasyahud adalah salah satu bacaan shalat yang **kaya** dengan empat keadaan i'rab. Anda baca minimal **5x/hari**.

\ayat{التَّحِيَّاتُ لِلَّهِ وَالصَّلَوَاتُ وَالطَّيِّبَاتُ}

\ayat{السَّلَامُ عَلَيْكَ أَيُّهَا النَّبِيُّ وَرَحْمَةُ اللَّهِ وَبَرَكَاتُهُ}

\ayat{السَّلَامُ عَلَيْنَا وَعَلَى عِبَادِ اللَّهِ الصَّالِحِينَ}

\ayat{أَشْهَدُ أَنْ لَا إِلَٰهَ إِلَّا اللَّهُ وَأَشْهَدُ أَنَّ مُحَمَّدًا رَسُولُ اللَّهِ}

Setelah shalat, kerjakan tabel ini bagian per bagian:

### Bagian 1 --- \ar{التَّحِيَّاتُ لِلَّهِ}

| Kata               | Harakat Akhir | Keadaan I'rab        | Mengapa? (kalau Anda tahu)        |
|--------------------|---------------|----------------------|------------------------------------|
| \ar{التَّحِيَّاتُ}  | dhammah       | rafa'                | mubtada                            |
| \ar{لِ}            | (harf jarr)   | --- (la mahalla lahu)|                                    |
| \ar{اللَّهِ}       | kasrah        | jarr                 | setelah harf jarr                  |

### Bagian 2 --- \ar{السَّلَامُ عَلَيْكَ}

| Kata             | Harakat Akhir | Keadaan I'rab |
|------------------|---------------|---------------|
| \ar{السَّلَامُ}  |               |               |
| \ar{عَلَى}       | (harf jarr)   | la mahalla lahu |
| \ar{ـكَ}         |               |               |

Catatan: \ar{ـكَ} adalah dhamir muttashil (kata ganti yang menempel) --- mabni, fi mahalli **jarr** karena setelah harf jarr.

### Bagian 3 --- \ar{مُحَمَّدًا رَسُولُ اللَّهِ}

| Kata                | Harakat Akhir | Keadaan I'rab        |
|---------------------|---------------|----------------------|
| \ar{مُحَمَّدًا}      |               | (\_\_\_\_) isim \ar{أَنَّ} |
| \ar{رَسُولُ}         |               | (\_\_\_\_) khabar \ar{أَنَّ} |
| \ar{اللَّهِ}         |               | (\_\_\_\_) mudhaf ilaih |

Refleksi: Anda baca tasyahud lengkap minimal 5 kali sehari. Apakah Anda baru kali ini **memperhatikan** keadaan i'rab tiap kata?

## Lab Tilawah: Identifikasi Empat Keadaan dalam Satu Halaman

Sesi tilawah Anda minggu ini di juz/surat apa? \_\_\_\_

Pilih satu halaman mushaf. Cari **satu contoh** untuk masing-masing dari empat keadaan i'rab:

| Keadaan I'rab | Kata yang Anda Temui | Surat:Ayat | Apa yang menandai? |
|---------------|----------------------|------------|---------------------|
| Rafa'         |                      |            |                     |
| Nashb         |                      |            |                     |
| Jarr          |                      |            |                     |
| Jazm          |                      |            |                     |

Tip: untuk menemukan **jazm**, cari kata yang didahului \ar{لَمْ}, \ar{لَمَّا}, atau \ar{لَا} (untuk larangan). Banyak ayat yang punya konstruksi ini, mis. \ar{لَمْ يَلِدْ وَلَمْ يُولَدْ} di Al-Ikhlas.

## Lab Doa Harian: Empat Keadaan dalam Satu Doa

Pilih SATU doa yang Anda baca rutin. Tulis lengkap, lalu identifikasi keadaan i'rab tiap kata yang **mu'rab**.

Contoh: doa keluar rumah:

\ar{بِسْمِ اللَّهِ تَوَكَّلْتُ عَلَى اللَّهِ لَا حَوْلَ وَلَا قُوَّةَ إِلَّا بِاللَّهِ}

| Kata                | Harakat Akhir | Keadaan I'rab |
|---------------------|---------------|---------------|
| \ar{بِسْمِ}          | (kasrah)      | jarr (setelah harf \ar{بِ}) |
| \ar{اللَّهِ}         | kasrah        | jarr (mudhaf ilaih)         |
| \ar{تَوَكَّلْتُ}     | (mabni)       | --- (fi'il madhi mabni)     |
| \ar{عَلَى}           | (harf jarr)   | la mahalla lahu             |
| \ar{اللَّهِ}         | kasrah        | jarr                         |
| \ar{لَا}             | (harf)        | la mahalla lahu             |
| \ar{حَوْلَ}          | fathah        | nashb (isim \ar{لَا})       |
| \ar{قُوَّةَ}         | fathah        | nashb                        |
| \ar{إِلَّا}          | (harf)        | la mahalla lahu             |
| \ar{بِاللَّهِ}       | kasrah        | jarr                         |

Doa Anda sendiri (kalau berbeda):

\_\_\_\_

| Kata | Harakat Akhir | Keadaan I'rab |
|------|---------------|---------------|
|      |               |               |
|      |               |               |
|      |               |               |
|      |               |               |

Refleksi: doa di atas mayoritas berisi kata **majrur** karena banyak harf jarr. Apa yang dikomunikasikan struktur seperti ini? (Hint: ketergantungan pada Allah, bersandar kepada-Nya.)

## Eksplorasi Mendalam: Al-Fatihah Ayat 7

\ayat{صِرَاطَ الَّذِينَ أَنْعَمْتَ عَلَيْهِمْ غَيْرِ الْمَغْضُوبِ عَلَيْهِمْ وَلَا الضَّالِّينَ}

Analisis lengkap (kerjakan di kertas terpisah, salin ke sini setelah selesai):

| Kata                | Harakat Akhir | Keadaan I'rab | Jabatan (kalau bisa)                 |
|---------------------|---------------|---------------|--------------------------------------|
| \ar{صِرَاطَ}         |               |               | badal dari \ar{الصِّرَاطَ} di ayat 6 → manshub |
| \ar{الَّذِينَ}       | (mabni)       | fi mahalli jarr| mudhaf ilaih (kata mawshul, mabni)  |
| \ar{أَنْعَمْتَ}      | (mabni)       | ---           | fi'il madhi + dhamir                 |
| \ar{عَلَيْهِمْ}      |               |               | (harf jarr + dhamir)                 |
| \ar{غَيْرِ}          |               |               | na'at untuk \ar{الَّذِينَ} → ikut jarr |
| \ar{الْمَغْضُوبِ}    |               |               | mudhaf ilaih → jarr                 |
| \ar{عَلَيْهِمْ}      |               |               |                                      |
| \ar{وَ}              |               |               | (harf athaf)                         |
| \ar{لَا}             |               |               |                                      |
| \ar{الضَّالِّينَ}    |               |               | ma'thuf, jarr (jamak mudzakkar salim) |

Sebagian tabel punya rafa', sebagian nashb, sebagian jarr. Hitung: rafa' = \_\_\_\_, nashb = \_\_\_\_, jarr = \_\_\_\_.

Catatan: ayat 7 adalah **doa lanjutan** dari ayat 6 \ar{اهْدِنَا الصِّرَاطَ الْمُسْتَقِيمَ}. Karena ayat 6 punya \ar{الصِّرَاطَ} dalam keadaan **nashb** (sebagai maf'ul bih dari \ar{اهْدِنَا}), \ar{صِرَاطَ} di ayat 7 sebagai badal juga **nashb**.
