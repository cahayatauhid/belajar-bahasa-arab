# Workbook Pelajaran 6 --- Pengertian I'rab

Workbook ini bukan PR yang harus dikejar selesai, melainkan **panduan untuk lebih sadar saat melakukan ibadah yang sudah rutin**. Yang berubah minggu ini: setiap ayat yang Anda baca, **mata Anda mulai memperhatikan harakat akhir tiap kata** dan bertanya: *kata ini sedang menjabat apa?* I'rab adalah penanda jabatan kata dalam struktur kalimat --- bukan "perubahan harakat", melainkan **rambu peran**.

> **Adab pengamatan**: saat shalat, fokus utama tetap khusyu' kepada Allah. Pengamatan i'rab dilakukan **setelah salam**, atau saat membaca ulang ayat di luar shalat untuk persiapan/refleksi.

## Refleksi Awal (5 Menit)

Jawab dari ingatan tanpa membuka catatan:

1. Sebutkan tiga jenis kalimah dan satu tanda untuk masing-masing.
2. Apa fungsi \ar{ال} pada \ar{الرَّجُلُ} dibanding \ar{رَجُلٌ}? (Pelajaran 5 bagian "Kekhususan")
3. Apakah harf bisa dimasuki tanwin? Mengapa atau mengapa tidak?

## Tantangan Vocabulary Mingguan

Target minggu ini: **kumpulkan 5 kata baru** dari bacaan shalat/tilawah Anda. Tetapi kali ini fokus tematik: **kata yang sama muncul dalam i'rab yang berbeda**.

Contoh kata yang sangat sering muncul di Quran dengan i'rab berbeda:

- \ar{اللَّهُ} (rafa') / \ar{اللَّهَ} (nashb) / \ar{اللَّهِ} (jarr) --- bisa Anda temui ketiganya
- \ar{رَبٌّ} dengan variasi i'rab
- \ar{إِنْسَانٌ} dengan variasi i'rab

### Catatan Vocabulary Anda

Kalau Anda menemukan kata yang sama di beberapa ayat dengan harakat akhir berbeda, catat:

| Kata Arab     | I'rab di sumber 1 | Sumber 1 | I'rab di sumber 2 | Sumber 2 |
|---------------|-------------------|----------|-------------------|----------|
|               |                   |          |                   |          |
|               |                   |          |                   |          |
|               |                   |          |                   |          |
|               |                   |          |                   |          |
|               |                   |          |                   |          |

Inti yang penting: **inti kata tidak berubah**, yang berubah hanya **harakat akhir** karena perbedaan jabatan.

## Lab Shalat: Memperhatikan Harakat Akhir di Al-Fatihah

Konteks: Anda baca Al-Fatihah minimal 17 kali sehari. Sekarang giliran **harakat akhir** yang diperhatikan.

### Misi 1 --- Al-Fatihah Ayat 2

\ayat{ٱلْحَمْدُ لِلَّهِ رَبِّ ٱلْعَٰلَمِينَ}

Setelah shalat, jawab:

| Kata                | Harakat Akhir       | Keadaan I'rab          | Mengapa?                                |
|---------------------|---------------------|------------------------|------------------------------------------|
| \ar{ٱلْحَمْدُ}      | dhammah \ar{ـُ}     | rafa' (marfu')         | mubtada → rafa'                         |
| \ar{لِ}             | (harf jarr)         | (harf, la mahalla lahu)| --- (harf tidak ber-i'rab)              |
| \ar{اللَّهِ}        | kasrah \ar{ـِ}      | jarr (majrur)          | setelah harf jarr → jarr                |
| \ar{رَبِّ}          | kasrah \ar{ـِ}      | jarr                   | na'at untuk \ar{اللَّهِ}, ikut majrur   |
| \ar{ٱلْعَٰلَمِينَ}  | (kasrah ya' mati)   | jarr                   | mudhaf ilaih → jarr (jamak mudzakkar salim, level 2) |

Catatan: \ar{ٱلْعَٰلَمِينَ} pakai bentuk khusus jamak mudzakkar salim (penanda i'rab dengan huruf, bukan harakat) --- akan didetailkan di Level 2.

### Misi 2 --- Al-Fatihah Ayat 3--4

\ayat{ٱلرَّحْمَٰنِ ٱلرَّحِيمِ ۝ مَالِكِ يَوْمِ ٱلدِّينِ}

Tanpa lookup, perhatikan harakat akhir:

| Kata                | Harakat Akhir   | Keadaan I'rab |
|---------------------|-----------------|---------------|
| \ar{ٱلرَّحْمَٰنِ}   |                 |               |
| \ar{ٱلرَّحِيمِ}     |                 |               |
| \ar{مَالِكِ}        |                 |               |
| \ar{يَوْمِ}         |                 |               |
| \ar{ٱلدِّينِ}       |                 |               |

Hampir semua kasrah! Kenapa? (Hint: ayat ini melanjutkan na'at + idhafah yang dimulai dari ayat 2 --- semua mengikuti jarr-nya \ar{اللَّهِ}.)

### Misi 3 --- Al-Fatihah Ayat 5

\ayat{إِيَّاكَ نَعْبُدُ وَإِيَّاكَ نَسْتَعِينُ}

Perhatikan harakat akhir \ar{نَعْبُدُ} dan \ar{نَسْتَعِينُ}:

- \ar{نَعْبُدُ}: harakat akhir = \_\_\_\_ → keadaan i'rab = \_\_\_\_
- \ar{نَسْتَعِينُ}: harakat akhir = \_\_\_\_ → keadaan i'rab = \_\_\_\_

Catatan: kedua fi'il mudhari ini dalam keadaan **rafa'** (akhir dhammah). Fi'il mudhari **bisa** ber-i'rab --- inilah yang membedakannya dari madhi dan amr yang mabni.

## Eksplorasi Mendalam: Pembalikan Jabatan dengan Penanda

Dari textbook Pelajaran 6:

\ar{ضَرَبَ زَيْدٌ عَمْرًا}

> "Zayd memukul Amr"

\ar{ضَرَبَ زَيْدًا عَمْرٌو}

> "Amr memukul Zayd"

Urutan kata **identik**. Yang berbeda: harakat akhir nama Zayd dan Amr.

Analisis:

| Kalimat 1: \ar{ضَرَبَ زَيْدٌ عَمْرًا}                                |
|----------------------------------------------------------------------|
| - \ar{زَيْدٌ}: dhammatain $\to$ rafa' $\to$ fa'il (subjek = Zayd)    |
| - \ar{عَمْرًا}: fathatain $\to$ nashb $\to$ maf'ul bih (objek = Amr) |
| Makna: **Zayd memukul Amr**                                          |

| Kalimat 2: \ar{ضَرَبَ زَيْدًا عَمْرٌو}                                |
|----------------------------------------------------------------------|
| - \ar{زَيْدًا}: fathatain $\to$ nashb $\to$ maf'ul bih (Zayd dipukul) |
| - \ar{عَمْرٌو}: dhammatain $\to$ rafa' $\to$ fa'il (Amr memukul)     |
| Makna: **Amr memukul Zayd**                                          |

Refleksi:

- Bahasa Indonesia tidak punya mekanisme ini. Kalau urutan diubah, makna ikut berubah. Apa **kelebihan** sistem Arab dibanding Indonesia di sini?

## Lab Dzikir: Tasbih, Tahmid, Takbir

Tiga dzikir paling tersebar setelah shalat:

\ar{سُبْحَانَ اللَّهِ}

\ar{الْحَمْدُ لِلَّهِ}

\ar{اللَّهُ أَكْبَرُ}

Identifikasi harakat akhir tiap kata:

| Dzikir              | Kata 1            | Kata 2          |
|---------------------|-------------------|-----------------|
| \ar{سُبْحَانَ اللَّهِ} | \ar{سُبْحَانَ}     | \ar{اللَّهِ}    |
| \ar{الْحَمْدُ لِلَّهِ}  | \ar{الْحَمْدُ}    | \ar{لِلَّهِ}    |
| \ar{اللَّهُ أَكْبَرُ} | \ar{اللَّهُ}      | \ar{أَكْبَرُ}   |

Tabel jawaban:

| Kata             | Harakat Akhir | Keadaan I'rab |
|------------------|---------------|---------------|
| \ar{سُبْحَانَ}    | fathah        | nashb         |
| \ar{اللَّهِ}      |               |               |
| \ar{الْحَمْدُ}    |               |               |
| \ar{لِلَّهِ}      |               |               |
| \ar{اللَّهُ}      |               |               |
| \ar{أَكْبَرُ}     |               |               |

Refleksi: Anda mengucapkan \ar{اللَّهُ أَكْبَرُ} **puluhan kali** setiap shalat (tiap perpindahan gerakan). Apakah Anda baru **menyadari** harakat akhirnya minggu ini?
