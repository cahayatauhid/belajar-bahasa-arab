# Workbook Pelajaran 5 --- Tiga Jenis Kalimah: Isim, Fi'il, Harf

Workbook ini bukan PR yang harus dikejar selesai, melainkan **panduan untuk lebih sadar saat melakukan ibadah yang sudah rutin**. Yang berubah minggu ini: setiap kata Arab yang Anda lafalkan, **mata Anda mulai mengkategorikannya secara otomatis** ke salah satu dari tiga: isim, fi'il, atau harf. Tidak ada kategori keempat.

> **Adab pengamatan**: saat shalat, fokus utama tetap khusyu' kepada Allah. Pengamatan klasifikasi kata dilakukan **setelah salam**, atau saat membaca ulang ayat di luar shalat untuk persiapan/refleksi.

## Tantangan Vocabulary Mingguan

Target minggu ini: **kumpulkan 5 kata baru** dari ibadah harian Anda, dengan **klasifikasi jenis kalimah** untuk masing-masing.

### Strategi Pengumpulan

**Strategi 1: Dari surat pendek shalat.**

Pilih satu surat pendek yang Anda hafal (Al-Ikhlas, Al-Falaq, An-Nas, Al-Kawthar, Al-'Ashr, dst.). Ambil 1--2 kata yang artinya belum sepenuhnya Anda pahami. Tentukan jenisnya.

**Strategi 2: Dari doa harian.**

Pilih doa yang sering Anda baca (doa makan, doa tidur, doa keluar rumah). Pilih kata yang asing.

**Strategi 3: Dari aplikasi kamus.**

Almaany atau lainnya. Lookup kata baru, lihat artinya, dan klasifikasikan.

Tip klasifikasi cepat:

- **Isim**: bisa dimasuki \ar{ال}, atau bisa menerima tanwin (\ar{ـٌ ـً ـٍ}), atau bisa didahului harf jarr.
- **Fi'il**: bisa didahului \ar{قَدْ}, \ar{س}, \ar{سَوْفَ}, atau punya akhiran \ar{ت} sukun (madhi mu'annats).
- **Harf**: tidak bisa keduanya. Definisi negatif --- "yang tidak punya tanda isim dan tidak punya tanda fi'il".

## Lab Shalat: Klasifikasi Kata di Al-Fatihah Ayat 5--7

Konteks: Al-Fatihah dibaca minimal **17 kali/hari** (5 waktu × 2--4 rakaat). Ayat 1--2 sudah dianalisis di textbook --- 7 isim, 3 harf, 0 fi'il. Sekarang giliran ayat 5--7 yang **mengandung fi'il**.

### Misi 1 --- Al-Fatihah Ayat 5

\ayat{إِيَّاكَ نَعْبُدُ وَإِيَّاكَ نَسْتَعِينُ}

Setelah shalat hari ini, jawab:

| Kata               | Jenis (isim/fi'il/harf) | Alasan                                  |
|--------------------|--------------------------|------------------------------------------|
| \ar{إِيَّاكَ}       |                          | (kata ganti $\to$ isim)                  |
| \ar{نَعْبُدُ}       |                          | (kata kerja "kami menyembah" $\to$ fi'il)|
| \ar{وَ}             |                          | (penghubung $\to$ harf)                  |
| \ar{إِيَّاكَ}       |                          |                                          |
| \ar{نَسْتَعِينُ}    |                          |                                          |

Total: \_\_\_\_ isim, \_\_\_\_ fi'il, \_\_\_\_ harf.

### Misi 2 --- Al-Fatihah Ayat 6--7

\ayat{اهْدِنَا الصِّرَاطَ الْمُسْتَقِيمَ ۝ صِرَاطَ الَّذِينَ أَنْعَمْتَ عَلَيْهِمْ}

Setelah shalat, klasifikasi 8 kata pertama:

| Kata               | Jenis | Alasan                              |
|--------------------|-------|-------------------------------------|
| \ar{اهْدِ}          |       | (fi'il amr, "tunjukilah")          |

| \ar{الصِّرَاطَ}     |       | (\ar{ال} $\to$ isim)                  |
| \ar{الْمُسْتَقِيمَ} |       |                                     |
| \ar{صِرَاطَ}        |       | (mudhaf; isim)                      |
| \ar{الَّذِينَ}      |       | (kata mawshul $\to$ isim)          |
| \ar{أَنْعَمْتَ}     |       | (fi'il madhi + dhamir)              |
| \ar{عَلَيْهِمْ}     |       | (\ar{عَلَى} = harf, \ar{ـهِمْ} = isim) |

Total: \_\_\_\_ isim, \_\_\_\_ fi'il, \_\_\_\_ harf.

Refleksi: Al-Fatihah ayat 1--2 (yang sudah dianalisis di textbook) tidak punya fi'il sama sekali. Ayat 5--7 kaya fi'il. Apa yang membedakan **fungsi** ayat-ayat tersebut? (Hint: 1--2 adalah pujian/deklarasi; 5--7 adalah doa permintaan.)
