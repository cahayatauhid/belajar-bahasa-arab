# Workbook Pelajaran 9 --- Fa'il

Workbook ini bukan PR yang harus dikejar selesai, melainkan **panduan untuk lebih sadar saat melakukan ibadah yang sudah rutin**. Yang berubah minggu ini: setiap kali Anda membaca atau mendengar **fi'il** di shalat/tilawah/dzikir, mata Anda mulai bertanya: **siapa fa'ilnya?** --- baik fa'il yang tampak sebagai isim eksplisit, maupun fa'il yang **mustatir** (tersembunyi dalam fi'il).

> **Adab pengamatan**: saat shalat, fokus utama tetap khusyu' kepada Allah. Pengamatan fi'il-fa'il dilakukan **setelah salam**, atau saat membaca ulang ayat di luar shalat untuk persiapan/refleksi.

## Refleksi Awal (5 Menit)

Jawab dari ingatan tanpa membuka catatan:

1. Apa beda jumlah ismiyyah dan jumlah fi'liyyah?

3. Sebutkan dua keadaan i'rab yang sudah dikenal dari Pelajaran 7.

## Tantangan Vocabulary Mingguan

Target minggu ini: **kumpulkan 5 fi'il madhi** dari bacaan shalat/tilawah Anda. Untuk masing-masing, identifikasi **siapa fa'ilnya** (eksplisit atau mustatir).

### Strategi Pengumpulan

**Strategi 1: Surat pendek shalat.**

Banyak surat pendek dimulai dengan fi'il madhi:

- Al-Kawthar (108): \ar{إِنَّآ أَعْطَيْنَٰكَ ٱلْكَوْثَرَ} (\ar{أَعْطَيْنَا} = "kami beri")
- Al-Ma'un (107): \ar{أَرَءَيْتَ ٱلَّذِى يُكَذِّبُ بِٱلدِّينِ} (\ar{أَرَءَيْتَ} = "tahukah engkau")
- Al-Fil (105): \ar{أَلَمْ تَرَ كَيْفَ فَعَلَ رَبُّكَ بِأَصْحَٰبِ ٱلْفِيلِ} (\ar{فَعَلَ} = "telah melakukan")

**Strategi 2: Doa harian.**

Banyak doa pakai fi'il madhi (terutama doa syukur).

**Strategi 3: Tilawah ayat-ayat sirah/kisah.**

Surat Al-Qashash, Yusuf, Maryam, dll. kaya dengan fi'il madhi cerita.

## Lab Shalat: Memetakan Fa'il di Surat-Surat Pendek

Konteks: surat pendek dibaca berulang setiap shalat. Mata Anda sudah hafal lafaz; minggu ini latihan **memetakan fa'il** untuk tiap fi'il.

### Misi 1 --- Surat Al-Fil 105:1

\ayat{أَلَمْ تَرَ كَيْفَ فَعَلَ رَبُّكَ بِأَصْحَابِ الْفِيلِ}

Setelah shalat, identifikasi:

| Bagian            | Fi'il          | Fa'il                          |
|-------------------|----------------|--------------------------------|
| \ar{أَلَمْ تَرَ}   | \ar{تَرَ}      | dhamir mustatir = "engkau" (\ar{أَنْتَ}) |
| \ar{فَعَلَ رَبُّكَ}| \ar{فَعَلَ}    | \ar{رَبُّكَ} (eksplisit, "Tuhanmu") |

Refleksi: ayat ini sangat tersebar di Indonesia (banyak yang hafal Surat Al-Fil dari kecil). Apakah Anda sebelumnya tahu bahwa \ar{رَبُّكَ} berperan **fa'il** dari \ar{فَعَلَ}?

\_\_\_\_

### Misi 2 --- Surat Al-Kawthar 108:1--2

\ayat{إِنَّآ أَعْطَيْنَٰكَ ٱلْكَوْثَرَ ۝ فَصَلِّ لِرَبِّكَ وَٱنْحَرْ}

Identifikasi:

| Bagian               | Fi'il          | Fa'il                                |
|----------------------|----------------|--------------------------------------|
| \ar{إِنَّآ أَعْطَيْنَا}  | \ar{أَعْطَيْنَا} | dhamir muttashil + mustatir = "Kami" |
| \ar{فَصَلِّ}         | \ar{صَلِّ}     | dhamir mustatir = "engkau"           |
| \ar{ٱنْحَرْ}         | \ar{ٱنْحَرْ}    |                                      |

Refleksi: ayat 1 = fi'il pertama (\ar{أَعْطَيْنَا}) dengan **fa'il "Kami"** (Allah). Ayat 2 = dua fi'il amr dengan **fa'il "engkau"** (Nabi). Apakah Anda merasakan **siapa berbicara kepada siapa** lebih jelas setelah memetakan fa'ilnya?

\_\_\_\_

### Misi 3 --- Surat Al-Fatihah Ayat 5--6

\ayat{إِيَّاكَ نَعْبُدُ وَإِيَّاكَ نَسْتَعِينُ ۝ اهْدِنَا الصِّرَاطَ الْمُسْتَقِيمَ}

Identifikasi:

| Bagian               | Fi'il         | Fa'il                |
|----------------------|---------------|----------------------|
| \ar{نَعْبُدُ}        | \ar{نَعْبُدُ} | dhamir mustatir = \_\_\_\_ |
| \ar{نَسْتَعِينُ}     |               |                      |
| \ar{اهْدِنَا}         |               |                      |

Catatan: \ar{اهْدِ} adalah fi'il amr, \ar{ـنَا} adalah dhamir muttashil sebagai maf'ul bih (objek "kami"). Fa'il dari amr adalah dhamir mustatir = "engkau" (Allah).

Refleksi: dalam ayat 5, **kami** (peserta shalat) adalah fa'il. Dalam ayat 6, **engkau** (Allah) adalah fa'il. Pembalikan ini menyampaikan dialog antara hamba dengan Tuhan. Pernah Anda merasakan dialog ini saat shalat?

## Eksplorasi Mendalam: Surat Yasin 36:38

\ayat{وَالشَّمْسُ تَجْرِي لِمُسْتَقَرٍّ لَّهَا}

> "Dan matahari berjalan menuju tempat berhentinya"

Analisis:

| Kata             | Jenis | Jabatan                              |
|------------------|-------|--------------------------------------|
| \ar{وَ}          | harf  | (athaf, la mahalla lahu)             |
| \ar{الشَّمْسُ}   | isim  | mubtada (jumlah ismiyyah baru)       |
| \ar{تَجْرِي}     | fi'il | mudhari, sebagai khabar              |
| Fa'il dari \ar{تَجْرِي} |       | dhamir mustatir \ar{هِيَ} merujuk ke \ar{الشَّمْسُ} |
| \ar{لِ}          | harf  | jarr                                 |
| \ar{مُسْتَقَرٍّ}  | isim  | majrur                               |
| \ar{لَّهَا}      | (\ar{لَ} harf jarr + \ar{ـهَا} dhamir muttashil)| (na'at terhadap \ar{مُسْتَقَرٍّ}) |

Refleksi:

- Mengapa fa'il dari \ar{تَجْرِي} **mu'annats** (\ar{هِيَ}, bukan \ar{هُوَ})?

  \_\_\_\_

- Fi'il \ar{تَجْرِي} di sini adalah **fi'il mudhari** sebagai khabar dari mubtada \ar{الشَّمْسُ}. Strukturnya = mubtada + khabar (fi'il), bukan jumlah fi'liyyah biasa. Mengapa demikian?

  \_\_\_\_ (Hint: kalimat dimulai dengan **isim** \ar{الشَّمْسُ}, bukan fi'il, jadi tetap jumlah ismiyyah.)

## Lab Quran: Fi'il Madhi yang Sangat Tersebar

Tiga fi'il madhi paling tersebar di Quran:

- \ar{قَالَ} (qala) = "berkata"
- \ar{خَلَقَ} (khalaqa) = "menciptakan"
- \ar{أَنْزَلَ} (anzala) = "menurunkan"

Cari satu ayat untuk masing-masing (boleh dari hafalan, mushaf, atau aplikasi), tunjukkan fi'il dan fa'ilnya:

| Fi'il          | Surat:Ayat | Fa'il (eksplisit / mustatir)        |
|----------------|------------|--------------------------------------|
| \ar{قَالَ}     |            |                                      |
| \ar{خَلَقَ}    |            |                                      |
| \ar{أَنْزَلَ}  |            |                                      |

Catatan: \ar{قَالَ} sering punya fa'il eksplisit (Musa, Ibrahim, Allah, dll.); \ar{خَلَقَ} fa'ilnya hampir selalu Allah; \ar{أَنْزَلَ} fa'ilnya Allah, tetapi sering pakai bentuk **majhul** \ar{أُنزِلَ} (akan dibahas Pelajaran 10).
